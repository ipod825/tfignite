Let the model set data related arguments.

Parameters
argsargparse.Namespace. The argument namespace to be set.
meta_infodict. Data related meta information that can be used to set args.

For example, a CNN model might need to define its archtecture based on the image height/width which is only know after the dataset is loaded in a training/evaluation script. The training script can then use this function to ask the model to modify args as needed. Such design frees users from modifying model-specific arguments in the training/evaluation script.
