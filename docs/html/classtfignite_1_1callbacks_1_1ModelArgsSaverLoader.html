Detailed Description

A callback that saves / loads the model arguments from the command line.

For first time training, we saves the model-specif arguments from the command line (defined by the model's add_parser function) into a file. For resumed training / evaluation, we check and ask user to confirm if saved arguments are different than those given in the command line.
